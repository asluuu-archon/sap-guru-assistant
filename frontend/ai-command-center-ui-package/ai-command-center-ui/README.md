# AI Command Center UI Framework

React + Vite dashboard UI package based on the supplied sample screens.

## Included screens
- Dashboard Overview
- Conversations
- Leads Management
- Pipeline Debugger
- AI Playground
- Business Brain
- Customer 360 View
- Reports & Analytics
- Automation
- Settings

## Run locally
```bash
npm install
npm run dev
```

Open the local Vite URL shown in the terminal.

## Build
```bash
npm run build
```

## Notes
This is a frontend UI framework with sample/static data. Replace the arrays in `src/main.jsx` with API calls when connecting to your backend.
